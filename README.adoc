= NTST Logger

This program is designed to record data from a geobuoy via a
MCC 128 DAQ to the storage device on a Raspberry Pi running
Raspbian (Debian 11, "Bullseye").

== DAQ

The DAQ is the only board installed on the Pi.

To get its address, run

[source, sh]
----
$ daqhats_list_boards
---

(the address is always 0).

This is tested on version:

[source, sh]
----
$ daqhats_version
daqhats version 1.4.0.8 is installed
---- 

=== DAQ Documentation.

See xref:https://mccdaq.github.io/daqhats/index.html[online].

=== DAQ Hardware Configuration

The coaxial cable is wired to channel 0 in and ground (AGND).
